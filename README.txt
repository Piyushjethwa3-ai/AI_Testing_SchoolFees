# School Fees Prediction

This project predicts school fees for the next 10 years using a simple linear regression model.

## Files:
1. school_fees_data.csv: Historical data for grades and fees.
2. school_fees_prediction.py: Script to train the model and predict future fees.
3. school_fees_model.pkl: Pre-trained linear regression model.
4. predicted_school_fees.csv: Predicted school fees for future years.
5. requirements.txt: Required libraries.
